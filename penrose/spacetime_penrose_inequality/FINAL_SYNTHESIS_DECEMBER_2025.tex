\section{The Fundamental Obstruction}
%==============================================================================

After extensive analysis, we identify the \textbf{true} obstruction:

\begin{tcolorbox}[colback=red!5!white, colframe=red!75!black, title=\textbf{THE GEOMETRIC OBSTRUCTION}]
For trapped surfaces, $H = \frac{1}{2}(\theta^+ + \theta^-) < 0$.

\textbf{Consequence:} Under \textbf{any} smooth outward flow $\partial_t\Sigma = \phi\nu$ with $\phi > 0$:
\begin{equation}
\frac{dA}{dt} = \int_\Sigma H\phi \, dA < 0
\end{equation}
Area \textbf{decreases} when moving outward from trapped surfaces.

This is not about the sign of $\mathrm{tr}_\Sigma k$---it's about the sign of $H$.
\end{tcolorbox}

\begin{remark}[Deeper Understanding]
The sign of $\mathrm{tr}_\Sigma k$ matters for the Jang equation method, but the true geometric issue is that trapped surfaces have $H < 0$. This causes:
\begin{enumerate}
    \item Area to decrease under outward deformation
    \item The natural flow direction to be \textbf{inward}, not outward
    \item Standard monotonicity formulas to fail
\end{enumerate}
\end{remark}

\section{What Would a Proof Need?}
%==============================================================================

A successful proof of the unconditional 1973 conjecture would need to either:

\subsection{Option 1: Find a Different Monotone Quantity}

Instead of area, find a functional $\mathcal{F}(\Sigma)$ such that:
\begin{enumerate}
    \item $\mathcal{F}(\Sigma_0) \leq \mathcal{F}(\Sigma^*)$ along some path from $\Sigma_0$ to MOTS $\Sigma^*$
    \item $\mathcal{F}(\Sigma^*) = A(\Sigma^*)$ for MOTS
    \item $\mathcal{F}(\Sigma_0) \geq A(\Sigma_0)$ for trapped $\Sigma_0$
\end{enumerate}

\textbf{Status:} No such functional is known.

\subsection{Option 2: Use Weak Solutions}

Allow ``jumps'' that increase area discontinuously, similar to Huisken-Ilmanen.

\textbf{Status:} The weak solution theory for the $\theta^+$-flow (or similar) is \textbf{OPEN}. This requires:
\begin{itemize}
    \item Level set formulation for $\theta^+ = 0$ condition
    \item Existence and uniqueness for degenerate PDE
    \item Monotonicity of appropriate mass functional
\end{itemize}

\subsection{Option 3: Spacetime Methods}

Use the 4D causal structure to bypass initial data obstructions.

\textbf{Status:} Promising but incomplete. Key gaps:
\begin{itemize}
    \item Lorentzian optimal transport requires SEC, but we only have DEC
    \item Null hypersurface methods have caustic singularities
    \item The connection between causal structure and mass is not fully understood
\end{itemize}

\subsection{Option 4: New Geometric Insight}

A completely new idea that we haven't thought of yet.

\textbf{Status:} This is what the field has been waiting for since 1973.
